# @electrosaur-labs/adobe

> **Superseded by [@electrosaur-labs/navigator](../reveal-navigator/).** Kept temporarily for reference.

Original Photoshop UXP command dialog for color separation. Implements the Mk 1.0 full-separation workflow as a single modal operation (analyze → posterize → separate → create layers).

The Navigator package provides the same production render plus real-time archetype exploration, palette surgery, interactive radar, and mechanical knobs. This package will be removed once Navigator feature parity is confirmed.
