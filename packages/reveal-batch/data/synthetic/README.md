# Synthetic Test Data

Programmatically generated Lab images targeting specific archetype centroids.
Used to fill fixture gaps for archetypes with no natural image coverage.

## Structure

- `8bit/` — Input 8-bit Lab PSDs (256x256, generated by `npm run generate:synthetic`)
- `output/` — Posterized output PSDs + JSON sidecars

## Generation

```bash
cd packages/reveal-batch
npm run generate:synthetic     # Generate input PSDs
npm run benchmark:synthetic    # Posterize to output
```

Do not populate manually — these are generated files.
