Separate
Controls
Screens 8
How many separate ink colors to extract. More screens = more fidelity but more setup cost. Typical range: 4–8 for commercial work, 8–12 for fine art reproduction.
Ghost Screen Removal 1.5%
Eliminate ghost screens — colors that cover less than this percentage of the image get merged into their nearest neighbor. Saves screens without visible quality loss.
Dust Removal 4px
Remove isolated pixel clusters smaller than this radius. Cleans up stray dots that would print as visible defects on mesh. Higher values = cleaner stencils but may lose fine detail.
Shadow Floor 8.0%
Set minimum ink density on each screen. Faint mask areas below this threshold get boosted to a printable level, preventing wash-out on press. 0% = no clamping.
Mesh Count
Thread count of your screen mesh. Higher TPI = finer halftone dots. Affects dithering scale so the pattern matches your actual mesh. Set to 0 for no mesh adjustment.
Trap Width 0pt
Expand lighter colors under darker neighbors to prevent white gaps from misregistration on press. Measured in points. 0 = no trapping.
Method
Separation Method
Standard: full Mk 1.5 engine with neutral sovereignty and highlight rescue. Adaptive: over-quantize then distil to target count. Hue-Aware: Mk 1.0 with hue gap recovery. Fast: Mk 1.0 without hue analysis. Classic: simple RGB median cut. Stencil: luminance-only for single-color work.
Color Mode
Color: full chromatic separation. B/W: two-tone black and white only. Grayscale: luminance-only separation across multiple gray levels.
Print Surface
Surface
What you're printing on. Auto detects from the image. White/Dark/Black set the substrate color explicitly. Transparent treats the substrate as see-through. Affects how background pixels are handled in the separation.
Tolerance 2.0
How close a pixel's color must be to the substrate before it's treated as background and skipped. Higher = more aggressive background removal. Lower = only exact matches are skipped.
Keep White
Guarantee a pure white screen in the palette. Essential when printing on dark substrates where white ink provides the base layer.
Keep Black
Guarantee a pure black screen in the palette. Keeps crisp outlines and text sharp instead of mixing from other dark colors.
Skip Transparent
Skip fully transparent pixels so they don't influence color selection. Turn off if your image uses transparency as a design element.
Output
Halftone
Dithering pattern to simulate gradients with solid inks. None = hard edges. Blue Noise = organic, film-like grain. Floyd-Steinberg = classic error diffusion. Bayer = ordered grid pattern.
Advanced
Method
Starting Palette
How median cut partitions color space. Detail Priority (variance) splits where error is highest — best for photographic work. Color Priority (median) splits where color count is highest — preserves more distinct hues in graphic art.
Color Matching
Distance formula for matching pixels to palette colors. CIE76: fast Euclidean — good for graphics. CIE94: chroma-weighted — better for photos with saturated colors. CIE2000: museum-grade perceptual accuracy — slow but best for fine art.
Color Selection
How the representative color is picked from each median cut bucket. Saliency: top 5% most vivid pixels — punchy results. Robust Saliency: same but clamps outliers — stable for warm-dominant images. Volumetric: weighted by pixel count — even coverage. Average: simple mean — flat but predictable.
Pre-Smoothing
Edge-preserving bilateral filter applied before separation. Smooths noise and compression artifacts while keeping hard edges sharp. Off for clean graphics, Light–Heavy for photos. Auto enables only when image entropy is high.
Gray Protection 0
When neutral (low-chroma) pixels exceed this % of the image, one palette slot is reserved for a dedicated neutral. Prevents white/gray backgrounds from consuming the chromatic color budget.
Saturation
Boost 1.4×
Multiplier that pushes centroid colors toward full saturation. Higher values rescue muted colors that median cut tends to average out. 1.0× = no boost.
Curve
Shape of the saturation boost curve. Linear: uniform boost across all chroma levels. Subtle–Aggressive: progressively stronger push on muted colors. Exponential: maximum rescue for the most desaturated pixels.
Gate —
Chroma threshold gate — pixels below this chroma level don't receive the saturation boost. Prevents noise in near-neutral areas from getting amplified. Higher = more selective boosting.
Color Merging
Merge Distance 6.0
Palette colors closer than this ΔE distance get merged into one screen. Higher values = fewer, more distinct colors. Useful for reducing screen count when similar inks would be hard to distinguish on press.
Auto Merge
Master switch for the palette reduction pass. When off, no colors are merged regardless of Merge Distance. When on, the engine applies the ΔE merge threshold.
Hue Recovery
Master switch for hue gap analysis. When on, the engine scans for hue sectors missing from the palette and injects rescue colors. Prevents median cut from dropping minority hues like green foliage in warm images.
Hue Lock 20°
Hue angle tolerance (in degrees) for locking recovered colors into their target sector. Wider angle = more forgiving hue matches. 0° = exact hue match required.
Color Priority
Lightness 1.2×
Weight for lightness (L*) axis in median cut splitting. Higher = more screens dedicated to tonal range. Lower = engine favors chromatic variety over tonal steps.
Color Intensity 2.0×
Weight for chroma (saturation) axis in median cut splitting. Higher = more screens dedicated to color intensity differences. Lower = engine favors tonal or hue variety.
Black Pull 3.0
Bias that pulls very dark pixels toward a dedicated black screen. Higher values = stronger black snap, giving crisper shadows and outlines. 0 = no black preference.
Tone
Highlight 90 L
L* value above which pixels are treated as highlights. Highlighted pixels can receive an extra saturation boost to prevent pastel wash-out in bright areas.
Highlight Boost 1.5×
Extra saturation multiplier applied to pixels above the Highlight Threshold. Rescues washed-out pastels in bright areas. 1.0× = no extra boost.
Shadow Point 15 L
L* value below which pixels are treated as deep shadows. Shadow pixels get special handling to preserve dark detail instead of collapsing to pure black.
Detail
Smoothing
Apply a median filter pass after separation to reduce color noise at screen boundaries. Smooths jagged edges between adjacent ink regions.
Fine Detail —
Counterbalances smoothing by restoring fine detail that the median filter might erase. Higher values = more detail preserved at the cost of noisier boundaries.
