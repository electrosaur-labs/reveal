## Parameter Category Architecture

Parameters are classified into mutually exclusive categories in SessionState:

| Category | Set | Behavior | Examples |
|----------|-----|----------|----------|
| **MECHANICAL_KNOBS** | 3 params | Fast path â€” mask re-render only | minVolume, speckleRescue, shadowClamp |
| **PRODUCTION_KNOBS** | 2 params | Production render only | trapSize, meshSize |
| **STRUCTURAL_PARAMS** | 25 params | Slow path â€” full re-posterize | targetColors, engineType, distanceMetric, ... |
| **UNIMPLEMENTED_KNOBS** | 5 params | UI exists, no engine reads | vibrancyThreshold, detailRescue, medianPass, maskProfile, neutralCentroidClampThreshold |

**ONE field flows through the system:** archetype `engine` â†’ ParameterGenerator `engineType` â†’ posterize() switch. No `engineMode` anywhere.

## Key Design Decisions

### Engine Unification: One Field, One Switch (2026-03-02)
Previously `engineType` (algorithm) and `engineMode` (routing wrapper) were two separate fields for one concept. Now `distilled` is just another case in posterize()'s switch. ProxyEngine has no special routing â€” it just calls posterize().

### Palette Consolidation: Commit Time Only (2026-03-02)
User edits can create near-duplicate palette entries. Consolidation runs ONLY in `exportProductionConfig()` (when Separate is clicked), never during preview. Only edited slots are candidates â€” engine-produced close colors are intentional.

### XMP Metadata: Separate Modal Required (2026-02-27)
`set fileInfo` inside `suspendHistory` silently drops. Metadata writes MUST be in a separate `executeAsModal` from layer creation.

### Palette Override: Knobs Before Override (2026-02-27)
ProxyEngine must apply knobs BEFORE palette override. minVolume's hue sector classification uses palette Lab values â€” if override changes the hue, sector rescue may fail and the swatch disappears.

### meshSize is Session-Level Equipment (2026-02-26)
Physical property of printing equipment, not archetype-driven. Default 230 TPI, survives archetype swaps.

### Production Render Uses Locked Palette (No Re-Posterize)
Palette curated via Surgery Mode. Re-posterizing would invalidate overrides.
