## Key Design Decisions

### UXP Keyboard Limitation (2026-02-23)
UXP non-modal dialogs (`dialog.show()`) do NOT receive keyboard events. Photoshop intercepts all keyboard input at the application level before it reaches the plugin. Only Escape works (handled by UXP dialog framework, not by `keydown` listener). All interactive features must use pointer events instead.

### Blink Comparator via Pointer Events (2026-02-23)
Click preview image to toggle original/posterized. Hold >300ms enters blink mode (alternates every 400ms). Disabled during highlight (swatch isolation). Original RGBA cached in `ProxyEngine._originalRGBA`, invalidated on new ingest.

### Interactive Radar Handles as HTML Overlays (2026-02-23)
Drag handles are `<div>` elements overlaid on the JPEG radar image, not rendered into the JPEG. This gives instant pointer response without waiting for JPEG re-encode. The debounce in `sessionState.updateParameter()` (50ms) throttles engine work.

### Posterize-on-Demand (2026-02-22)
Only top 3 archetype cards get eagerly posterized. The rest posterize lazily when clicked. With 25 archetypes, eager posterization of all would add several seconds to initial load.

### PosterizationEngine Decomposition (2026-02-21)
Split monolithic 4693-line engine into 5 focused modules. Orchestrator delegates to CentroidStrategies, HueGapAnalyzer, PaletteOptimizer, MedianCutPartitioner.

### Externalized Engine Tuning (2026-02-17)
All tunable parameters (lWeight, cWeight, vibrancyBoost, chromaGate, etc.) moved from hardcoded values into archetype JSON files. Each archetype defines its own tuning profile.

### DNA-Driven Adaptive Color Count (2026-02-17)
`targetColors` no longer hardcoded. Archetype DNA determines optimal color count based on image complexity.

### Palette Override Pipeline (2026-02-14)
Order in `updateProxy`: Restore baseline → Apply palette override → Apply knobs. Overrides indexed against baseline. `_buildOverriddenPalette` reads from `_baselineState.palette`.

### MinVolume Preserves Index Stability (2026-02-14)
Weak colors remapped in colorIndices, palette array length unchanged. Override indices stay valid.

### Production Render Uses Locked Palette (No Re-Posterize)
Palette curated via Surgery Mode. Re-posterizing would invalidate overrides.

### Proxy Preview Uses Mask-Based Rendering
Preview from masks (not raw colorIndices) so speckleRescue effects are visible.

### Color Picker Cancel Detection (2026-02-15)
UXP `showColorPicker` batchPlay doesn't reliably return `_obj:"cancel"` on dismiss. Instead, compare foreground color before/after — Photoshop reverts it on cancel.

### Swatch Deletion (2026-02-15)
Alt+click merges deleted swatch's pixels into nearest CIE76 neighbor. Swatch shows dimmed + red X badge. Revert restores it. Deletion stored in paletteOverrides as `{deleted: true}`.
