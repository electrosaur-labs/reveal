## Key Design Decisions

### Production Render Uses Locked Palette (No Re-Posterize)
The palette was curated via Surgery Mode — merged, overridden, and accepted by the user. Re-posterizing at full resolution would produce different centroids, invalidating palette overrides (stored by index). ProductionWorker takes the locked palette as-is and runs **separation only** at full resolution.

### Proxy Preview Uses Mask-Based Rendering
After knob application, preview is generated from masks (not raw colorIndices). This ensures speckleRescue effects are visible. Despeckled pixels absorb into surrounding color (not white) because in print they'd merge with adjacent ink.

### Navigator is a Command Dialog (Not Panel)
Changed from persistent panel to Plugins → Navigator... command that opens a non-modal dialog. Dialog auto-dismisses after production render. Matches reveal-adobe pattern.
