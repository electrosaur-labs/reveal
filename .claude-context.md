## Architecture Notes

### DNA v2.0 Flow
```
1. Image (Lab PSD) → DNAGenerator.generate()
   └─> DNA v2.0 { version, global, sectors, dominant_sector }

2. DNA v2.0 → ArchetypeLoader.matchArchetype()
   ├─> ArchetypeMapper.getBestMatch() [40/45/15 scoring]
   └─> Returns archetype with matchScore/matchBreakdown

3. Archetype + DNA → ParameterGenerator.generate()
   └─> Configuration with meta.matchScore/matchBreakdown

4. Configuration → PosterizationEngine + SeparationEngine
   └─> Posterized PSD + JSON sidecar
```

### Backward Compatibility
- DNA v1.0 images (no `version` field) use legacy 4D weighted distance
- Legacy fields preserved at top level in DNA v2.0
- Mixed datasets (v1.0 + v2.0) supported in same batch
