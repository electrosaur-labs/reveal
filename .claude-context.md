## Parameter Category Architecture

Parameters are classified into mutually exclusive categories, declared in core (`ParameterGenerator.CONFIG_CATEGORIES`):

| Category | Set | Behavior | Examples |
|----------|-----|----------|----------|
| **MECHANICAL** | 3 params | Fast path — mask re-render only | minVolume, speckleRescue, shadowClamp |
| **PRODUCTION** | 2 params | Production render only | trapSize, meshSize |
| **STRUCTURAL** | 25 params | Slow path — full re-posterize | targetColors, engineType, distanceMetric, ... |
| **UNIMPLEMENTED** | 5 params | UI exists, no engine reads | vibrancyThreshold, detailRescue, medianPass, maskProfile, ditherType |

**Knob defaults** declared in core (`ParameterGenerator.KNOB_DEFAULTS`):
- MECHANICAL: `{ minVolume: 0, speckleRescue: 0, shadowClamp: 0 }`
- PRODUCTION: `{ trapSize: 0, meshSize: 230 }`

Navigator imports these — no local copies.

## Key Design Decisions

### Shared Pixel Processing Utility (2026-03-02)
`pixelProcessing.js` in `src/utils/` holds shared constants (`DIM_COLOR`, `BG_COLOR`) and the `applySuggestionGhost()` pixel loop. Both SessionState (proxy preview) and Loupe (native-res tile) delegate to it. No more duplicate 47-line ghost loops.

### Config Single Source of Truth in Core (2026-03-02)
`CONFIG_CATEGORIES`, `KNOB_DEFAULTS`, and `extractMechanicalKnobs()` live in `ParameterGenerator`. Navigator imports and delegates rather than maintaining its own copies.

### Navigator→Core API Discipline (2026-03-02)
Navigator must use core's public API (`Reveal.*`) for all color math. No inline reimplementations of distance metrics or color conversions.

### Engine Unification: One Field, One Switch (2026-03-02)
`distilled` is just another case in posterize()'s switch. No `engineMode` anywhere. ProxyEngine has no special routing.

### Palette Consolidation: Commit Time Only (2026-03-02)
Near-duplicate palette entries consolidated only in `exportProductionConfig()`. Never during preview.

### meshSize is Session-Level Equipment (2026-02-26)
Physical property of printing equipment, not archetype-driven. Default 230 TPI.
